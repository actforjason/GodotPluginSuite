## Addons
* Quickly Create Node

  Middle click to both select node and open Create New Node dialog.

  Grab focus on scene tree instead of viewport after creating a new node.


* Right Align Tab Bar

  Aligns the tab bar of `EditorSceneTabs` and `EditorBottomPanel` to the right side of the editor window.

* Only Run

  Only run the project, without building it. Middle click to use console or stop running.
