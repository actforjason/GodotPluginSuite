## Addons
* Quickly Create Node

  Middle click to both select node and open Create New Node dialog.

	Auto enter naming mode after creating a new node, and Middle click to submit the name and open the Create New Node dialog again.

* Right Align Tab Bar

  Aligns the tab bar of `EditorSceneTabs` and `EditorBottomPanel` to the right side of the editor window.

* Only Run

  Add a button to only run the project, without building it. Middle click to use console or stop running.
